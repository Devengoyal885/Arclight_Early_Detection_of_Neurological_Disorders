# Roadmap

## ✅ v1.0.0 — Foundation (Completed)
- [x] PyTorch models: ResNetMRI (binary) + CNN2D (multiclass)
- [x] Flask web interface (original)
- [x] MRI preprocessing pipeline

## ✅ v1.1.0 — Platform Transformation (Current)
- [x] FastAPI backend with REST endpoints
- [x] React + Vite + TypeScript + Tailwind frontend
- [x] Drag & drop multi-image upload
- [x] Animated MRI scanner UI
- [x] Prediction result cards with confidence scores
- [x] Probability distribution visualization
- [x] Risk meter gauge
- [x] Multi-scan batch analysis
- [x] Explainability module (simulated heatmap)
- [x] PDF report generation
- [x] Medical disclaimer compliance
- [x] GitHub CI/CD workflows
- [x] Complete documentation suite

## 🚧 v1.2.0 — Enhanced Analysis (Q3 2025)
- [ ] DICOM file format support
- [ ] NIfTI (.nii.gz) file support
- [ ] Real Grad-CAM implementation via PyTorch hook
- [ ] 3D volume visualization (mid-slice slider)
- [ ] Improved model accuracy with larger dataset

## 🔮 v1.3.0 — Authentication & History (Q4 2025)
- [ ] Firebase authentication (Google, Email)
- [ ] Scan history per user
- [ ] Previous scans comparison view
- [ ] Cloud storage for scan uploads

## 🚀 v2.0.0 — Production Ready (Q1 2026)
- [ ] HIPAA-compliant data handling
- [ ] Multi-language support (Hindi, Spanish, French)
- [ ] Mobile app (React Native)
- [ ] Wearable device integration
- [ ] Telemedicine referral integration
- [ ] EHR/EMR API connectors

## 🔬 Research Track
- [ ] Transformer-based vision model (ViT)
- [ ] Self-supervised pre-training on brain MRI
- [ ] Multi-modal analysis (MRI + clinical data)
- [ ] Longitudinal tracking (repeat scans)
